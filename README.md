# Socio-Political Protests Iran Data (SPPI)

## Project Overview
This project extracts and codes socio-political protest data from Iranian news agencies using AI-assisted coding.

## Current Status
- ✅ Data imported from ILNA
- 🔄 LLM coding in progress
- ⏳ Website development pending

## Setup
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/iran-protest-data.git

# Navigate to project
cd iran-protest-data

# Create virtual environment
python -m venv venv

# Activate it
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set up API key
echo "OPENAI_API_KEY=your_key_here" > .env
